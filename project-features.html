Snap Scheduling App

Project Proposal

Snap is a mobile application that tries to improve the short comings of the existing and previously used scheduling systems.

Traditionally, people used different methods to schedule and manage their time and tasks. This includes memorizing of tasks in their mind (which is still done by a significant number of people), while others use paper to write down their schedules.

The proposed system is an attempt to provide a solution for most of the shortcomings of the existing systems mentioned in the previous sections. The main innovation of this project is the importance it places on the dynamic nature of students’ lives.

The app has been advertised as “forgiving” and “inclusive” because it is built to seamlessly accommodate changing schedules, sudden interruptions, shifting priorities, etc.

The Proposed System

Google's Android has by far the largest share of the mobile OS market in Ethiopia, being used in around 90 percent of the smartphones in the country. To accommodate these different devices, performant tools and programming paradigms must be used.

The entire app is self-contained. Although an interface between our project and mainstream calendar apps might be needed to allow users to integrate their existing data.
